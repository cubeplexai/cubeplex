# 欢迎使用 CubePlex

CubePlex 是一个全栈 AI 代理工作区。你可以在一个平台中使用多模型对话、可安装技能、持久记忆、MCP 工具集成和任务自动化。

## 你可以做什么

- **与 AI 代理对话**——使用来自 Anthropic、OpenAI 和自定义端点的模型。你可以附加文件、执行代码并接收结构化制品。
- **安装技能**来扩展代理的能力——从内置功能到社区注册表中的技能。
- **构建持久记忆**，让代理在不同对话间持续记住信息，并按你选择的层级共享（个人、工作区或全组织）。
- 通过模型上下文协议（MCP）连接器**连接外部工具**——数据库、API、SaaS 产品等。
- 使用计划运行（cron、间隔、一次性）和事件驱动触发器（启动代理运行的 webhook）**自动化任务**。
- 通过组织和工作区角色、模型访问策略和成本跟踪**管理团队**。

## 部署选项

<Tabs groupId="deploy-mode">
<TabItem value="cloud" label="云端">

</TabItem>
<TabItem value="self-hosted" label="自托管">

将 CubePlex 部署在自己的基础设施上——安装步骤见[部署指南](./deployment/overview.md)。首位注册用户会进入一次性的**引导设置**向导，为组织命名、选择 slug 并创建第一个工作区，同时成为组织所有者。后续用户会作为成员加入该组织，并各自拥有个人工作区。

</TabItem>
</Tabs>
