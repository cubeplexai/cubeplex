# 个人资料设置

个人资料页面用于管理你自己的账户：头像、显示名称、密码、API 密钥和账户删除。可从侧边栏的 **头像菜单** 打开（点击头像，然后选择 **个人资料设置**）。

## 头像

你的头像会显示在对话、成员列表和侧边栏中你的姓名旁边。你可以上传照片，也可以选择生成的头像。

- **上传** — 点击 **上传** 并选择图片文件。CubePlex 会将其转换为 256×256 的方形 PNG（保持宽高比并居中）后存储。任何常规图片均可；人脸和徽标在头像尺寸下都能良好显示。
- **随机生成** — 点击 **随机生成**，生成一组包含 30 个卡通头像的新网格（DiceBear “notionists”风格），然后点击图库中的任一个将其设为头像。每次随机生成都会产生新的随机集合，因此你可以不断生成，直到找到喜欢的头像。
- **重置** — 点击 **重置** 清除选择并恢复为默认生成头像。这会移除已上传的照片或已选择的生成头像，使头像回退为根据你的账户种子自动生成的头像。

当前头像显示在编辑器左上角，三个操作按钮旁边。编辑器完成保存后，你的选择会同步到所有显示头像的位置——群聊发送者徽章、成员列表和侧边栏。

![个人资料头像编辑器，显示图库和上传控件](/img/settings/avatar-editor.png)

## 其他设置

个人资料页面的其余部分包含其他账户字段：

- **显示名称** — 其他成员在你的消息和成员列表中看到的名称。
- **密码** — 更改登录密码。
- **API 密钥** — 创建用于以编程方式访问 CubePlex API 的密钥。
- **删除账户** — 永久移除账户和个人数据。
