# 使用记忆

开始使用记忆无需配置任何内容。智能体会在你聊天时自动回忆相关记忆并存储新记忆。本页面介绍如何主动塑造智能体所记住的内容。

## 教导智能体

创建记忆最直接的方式是告诉智能体记住某些内容：

> “请记住，我喜欢简洁的回答，并附带代码示例。”

这会成为你个人记忆中的一项 **preference**，并应用于你的所有工作区。

默认情况下，智能体会保存到 **个人** 范围。除非你另行决定，这可以确保记忆仅对你自己可见。因此，即使项目事实没有额外指示，也会保存到你的个人记忆中：

> “请记住，我们的 API 对所有响应字段使用 snake_case。”

智能体会将此内容作为个人记忆中的 **project_fact** 保存。

如需将记忆分享给整个团队或组织，请明确提出要求。只有在你明确说明时，智能体才会写入工作区或组织范围：

> “为整个工作区保存这条信息：我们的 API 对所有响应字段使用 snake_case。”

> “将此保存为组织范围的政策：所有面向客户的文本在发布前都必须经过内容团队审核。”

第一条会成为工作区中的 **project_fact**；第二条会成为组织记忆中的 **org_policy**。

## 纠正智能体

当智能体出错时，直接纠正它：

> **智能体：** “我会使用 npm 设置项目……”
>
> **你：** “我们使用 pnpm，不使用 npm。”

智能体会将一条 **correction** 保存到你的个人记忆中。在未来的对话中，它会使用 pnpm，而无需你再次说明。如果希望整个团队继承此更正，请告诉智能体分享它（“将其保存到工作区”）。

更正可以在任意范围内生效，个人更正会随你应用到所有工作区：

> “这是错的——我偏好深色模式的代码块，而不是浅色模式。”

## 智能体自动记住的内容

除明确指示外，智能体在对话中识别出可复用信息时，也可能存储记忆条目。例如：

- 你逐步描述部署流程——智能体可能将其保存为 **procedure**。
- 你与智能体就某种方案达成一致——它可能将结果存储为 **decision**。
- 你分享智能体此前不知道的事实——它可能将其记录为 **project_fact**。

每条自动创建的记忆都包含置信度分数。来自明确指示（“请记住……”）的条目通常比智能体从上下文推断的条目具有更高置信度。

## 记忆的实际应用

以下是记忆如何逐渐积累并长期发挥作用的实际示例：

1. **第 1 天** — 你告诉智能体：*“将此保存到工作区：我们的后端使用 FastAPI 和 PostgreSQL，部署在 AWS ECS 上。”* 智能体会保存一条工作区范围的 project_fact，以便整个团队继承。
2. **第 2 天** — 一位团队成员请求智能体帮助编写数据库迁移。智能体已知技术栈为 PostgreSQL，因此无需询问即可使用正确语法。
3. **第 3 天** — 你纠正智能体：*“我们使用 Alembic 进行迁移，不使用原始 SQL——也将此保存到工作区。”* 智能体会将一条 correction 保存到工作区记忆中。
4. **第 4 天** — 另一位团队成员请求协助完成新的迁移。智能体会回忆 PostgreSQL 事实和 Alembic 更正，并生成 Alembic 迁移文件。

## 选择范围

智能体始终将新记忆默认保存到 **个人** 范围，并根据内容推断其 *类型*。只有在你明确要求分享时，它才会写入工作区或组织范围：

| 内容模式 | 默认范围 | 常见类型 |
|---|---|---|
| “我偏好……”/“我喜欢……” | 个人 | preference |
| “我们的项目使用……”/“代码库……” | 个人 | project_fact |
| 纠正错误 | 个人 | correction |
| 就方案达成一致 | 个人 | decision |
| 描述分步骤流程 | 个人 | procedure |
| “将此保存到工作区：……” | 工作区 | project_fact / procedure / decision |
| “将此保存为组织范围的政策：……” | 组织 | org_policy |

如需将记忆提升为你个人以外的范围，请在要求智能体记住它时说明（“将其保存到整个工作区”）。你无法在记忆中心更改现有条目的范围，因此请在一开始就明确说明。

## 提示

- **具体说明。** “请记住，我们在 Python 文件中使用 4 空格缩进”比“记住我们的代码风格”更有用。
- **尽早纠正。** 越早修正错误，智能体在其他对话中重复该错误的可能性就越低。
- **定期查看记忆中心。** 随着时间推移，记忆条目可能过时。归档陈旧条目可让智能体上下文保持整洁。
