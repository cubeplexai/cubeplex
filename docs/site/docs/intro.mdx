# Welcome to CubePlex

CubePlex is a full-stack AI agent workspace. You get multi-model conversations, installable skills, persistent memory, MCP tool integrations, and task automation — all in one platform.

## What you can do

- **Chat with AI agents** across models from Anthropic, OpenAI, and custom endpoints. Attach files, execute code, and receive structured artifacts.
- **Install skills** to extend what your agent can do — from built-in capabilities to community registries.
- **Build persistent memory** that the agent recalls across conversations, shared at the level you choose (personal, workspace, or org-wide).
- **Connect external tools** via Model Context Protocol (MCP) connectors — databases, APIs, SaaS products.
- **Automate tasks** with scheduled runs (cron, interval, one-shot) and event-driven triggers (webhooks that kick off agent runs).
- **Manage your team** with organization and workspace roles, model access policies, and cost tracking.

## Deployment options

<Tabs groupId="deploy-mode">
<TabItem value="self-hosted" label="Self-hosted" default>

Deploy CubePlex on your own infrastructure — see the [Deployment guide](./deployment/overview.md) for installation instructions. The first user to register is taken through a one-time **onboarding** wizard to name the organization, choose a slug, and create their first workspace, becoming the organization's owner. Subsequent users join that organization as members, each with their own personal workspace.
